"Automata".

Este proyectorealiza la lectura de las denomindas palabras (Universo binario), esto desde un documento con extension ".txt" donde con base a diferentes criterios este decide si las palabras son aceptadas o rechazadas.
Posteriormente las palabras con almacenadas en documentos de texto con extension ".txt" con base en la clasificación que se les haya asigando. Ver más.

"Teorema de Moivre".

Proyecto el cual nos permite calcular las raices de un número complejo, siendo la cantiidad de raices a calcular introducidas por el usuario, esto utilizando el teorema de Moivre.
También este nos deplegara una nueva ventana en donde se graficaran las raices obtenidas en el calculo.
Ver más.

"Universo de Números binarios".

Unierso conformado por numeros binarios, donde la cantidad de digitos esta dado por la variable "K" siendo así una familia desde 0 hasta K, posteriormente se genera la familia del tamaño indicado y se guaradan en un archivo con extension ".txt".
